More features

Everything in one place

Kaisho covers the full freelance workflow — from inbox capture to scheduled AI automation.

Inbox & Capture

Capture anything fast. kai inbox add drops items into your inbox for triage later — via CLI or web.

Knowledge Base

Searchable notes and research files. Browse the tree, preview markdown, or kai kb search from the terminal.

GitHub Integration

Pull issues and PRs per customer. Grouped by repo, visible in web and CLI without leaving your workflow.

Scheduled Automation

Define cron jobs backed by prompts. Kaisho runs them on schedule and writes AI-generated output back to your files.

Multi-User Profiles

Multiple BACKEND configurations. Switch contexts between clients or projects without friction.

Command Palette

Every feature is a kai subcommand. Tab-complete your way through tasks, clocks, customers, knowledge, and more.

CLI-first

Everything reachable from the terminal.

The web UI is optional. Every feature in Kaisho is a kai subcommand. Start timers, query your AI advisor, manage tasks — all without leaving the shell.

$ kai briefing
Morning briefing — 4 open tasks, 2 customers active $ kai clock start "Beta GmbH sprint review" --customer beta-gmbh
Started: 09:14 — Beta GmbH sprint review $ kai ask "Am I on track with Beta GmbH?"
You have used 89% of the Beta GmbH budget... $ kai clock book --customer beta-gmbh --hours 2.5
Booked 2.5 h to Beta GmbH / Contract-Q2 $ kai task add "Review invoice draft" --customer acme
Added task #47 — Review invoice draft [Acme Corp] $ kai kb search "oauth"

Getting started

How it works

Kaisho is a Python package. Three steps from install to work.

01
Install
pip install kaisho then kai serve to start the local server.
02
Point to your files
Set your org-mode or markdown directory in config. Kaisho reads and writes your existing plain text files.
03
Work
Open localhost:8765 or use kai in any terminal. Your data stays on your machine.
