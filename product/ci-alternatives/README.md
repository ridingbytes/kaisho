# CI Alternatives Archive

All explored logo concepts, color palettes, and design
variations created during the Kaisho branding process.

## Final Choice

- Logo: Chevron Dot (Graphite Lavender)
- Palette: Graphite Lavender
- Files in use: ../website/kaisho-logo*.svg, ../website/kaisho-wordmark*.svg


## Directory Structure

### logos-enso-teal/
First iteration. Enso (brush-stroke circle) in SENAITY
teal (#75A4AA). Rejected: too similar to SENAITY branding.

### logos-enso-indigo/
Second iteration. Enso circle in indigo (#818cf8) matching
the original dark app theme. Rejected: generic, indigo
palette was replaced.

### logos-chevron-dot-warmstone/
Chevron Dot logo in Warm Stone colors (#292524). The
chevron-dot concept was selected but the Warm Stone palette
had insufficient label contrast.

### logos-chevron-dot-graphite/
FINAL. Chevron Dot logo in Graphite Lavender (#1e1e2e).
This is the version deployed to the app and website.

### logo-concepts/
Interactive HTML pages showing all 6 logo concepts explored:
1. The Prompt (chevron + cursor)
2. The K-Mark (abstract K)
3. Four Tiles (2x2 grid)
4. Column Stack (3 bars)
5. Chevron Dot (CHOSEN)
6. Rounded K

### palettes/
Interactive HTML pages with palette explorations:
- v1: 6 bright accent palettes (Arctic, Ink+Flame,
  Violet Depth, Forest, Pure+Signal, Mist+Ocean)
- v2: 6 neutral palettes with kanban mocks (Warm Stone,
  Cool Slate, Graphite Lavender, Sand+Navy, Zinc,
  Pearl+Deep Teal)
- color-system-warmstone: full design system spec for
  Warm Stone (superseded by Graphite Lavender)
