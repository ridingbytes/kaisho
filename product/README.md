## Key Decisions Made

- **Name:** Kaisho (開手, "open hand")
- **Domain:** kaisho.dev
- **CLI command:** `kai`
- **Logo:** Open Bracket Palm (Three Lines)
- **Palette:** Graphite Lavender (light-first)
- **Monetization:** Hybrid (Phase 1: license keys,
  Phase 2: Tauri desktop, Phase 3: hosted tier)
- **Pricing:** Free / EUR 5 Personal / EUR 19 Pro
- **Company:** Riding Bytes (also SENAITY)
