## Key Decisions

- **Name:** Kaisho (開手, "open hand")
- **Logo:** Open Bracket Palm (Three Lines)
- **Palette:** Zinc (true neutral gray)
- **Architecture:** Single-user, multiple profiles
- **Backends:** org-mode, Markdown, JSON, SQL
- **Monetization:** Free open source + paid tiers
- **Domain:** kaisho.dev
- **CLI:** `kai`
